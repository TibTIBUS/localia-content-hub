# BE-005 - Calendrier editorial Localia

## Planning retenu

ChatGPT cree les contenus :
- lundi matin a 05:30
- jeudi matin a 05:30

Hermes publie :
- lundi a 10:00
- jeudi a 10:00

## Logique editoriale

Deux publications par semaine permettent de garder une presence reguliere sans saturer l audience.

## Alternance recommandee

### Lundi

Objectif : contenu educatif ou probleme-solution.

Themes prioritaires :
- SEO local
- Google
- site internet
- visibilite
- diagnostic

Templates recommandes :
- T01 probleme-solution
- T02 checklist

### Jeudi

Objectif : contenu commercial, offre, IA ou reseaux sociaux.

Themes prioritaires :
- gestion Facebook et Instagram
- IA et automatisation
- application web sur mesure
- formation IA
- offre Localia

Templates recommandes :
- T03 offre
- T01 probleme-solution
- T02 checklist

## Regle anti-doublon

Ne pas traiter le meme theme deux publications de suite.

## Rotation type sur 4 semaines

Semaine 1 :
- Lundi : SEO local
- Jeudi : reseaux sociaux

Semaine 2 :
- Lundi : site internet
- Jeudi : IA et automatisation

Semaine 3 :
- Lundi : Google et confiance
- Jeudi : applications web sur mesure

Semaine 4 :
- Lundi : checklist visibilite
- Jeudi : offre ou diagnostic
