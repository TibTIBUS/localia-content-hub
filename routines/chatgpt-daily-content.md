# Routine ChatGPT - creation quotidienne de contenus Localia

Frequence recommandee : tous les jours a 05:30 Europe/Paris.

## Mission

Creer un contenu social media Localia pret a publier sur Facebook et Instagram, puis le stocker dans ce depot GitHub.

## Etapes

1. Choisir un sujet utile pour les commercants, artisans, PME locales ou associations.
2. Verifier que le sujet correspond aux services Localia : visibilite locale, site internet, SEO local, Google, reseaux sociaux, IA, automatisation, acquisition client.
3. Rediger une version Facebook claire, professionnelle et orientee resultats.
4. Rediger une version Instagram plus courte, plus dynamique, avec hashtags.
5. Creer un brief visuel coherent avec les visuels Facebook Localia existants.
6. Ajouter un fichier JSON dans `/content/ready`.
7. Ajouter un brief visuel dans `/assets/visual-briefs`.
8. Ajouter un visuel source dans `/assets/images` si possible.
9. Ne jamais placer un contenu en `ready` si le contenu est incomplet.

## Regle visuelle critique

Ne pas inventer de nouveau style. Les visuels doivent rester dans la meme case graphique que les publications Facebook Localia existantes.

## Statut

Le contenu cree par la routine doit etre en `ready` uniquement si :
- le texte Facebook existe ;
- la caption Instagram existe ;
- le visuel ou brief visuel existe ;
- la date de publication est definie ;
- le contenu n est pas un doublon recent.
