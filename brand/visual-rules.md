# BE-001 - Regles visuelles Localia

## Format

Formats prioritaires :
- 1080 x 1080 pour Facebook et Instagram.
- 1080 x 1350 pour formats verticaux premium.

## Identite

Les visuels Localia sont des mini-infographies premium.
Ils doivent etre comprehensibles rapidement, meme sur smartphone.

## Elements recurrents

- Fond vert sombre ou fond clair structure.
- Accents orange sur les mots importants.
- Gros titre tres lisible.
- Blocs arrondis.
- Illustration centrale premium.
- CTA orange ou signature forte.
- Bandeau bas possible avec services.

## Style illustration

Deux familles sont autorisees :

1. Illustration IA premium : smartphone, ordinateur, dashboard, Google, Facebook, IA, robot, carte locale.
2. Photo realiste premium : commerce, restaurant, artisan, situation locale, objet metier.

## Densite

Un visuel doit contenir :
- 1 idee principale ;
- 1 titre fort ;
- 1 visuel principal ;
- 2 a 5 blocs maximum ;
- 1 CTA ou signature.

## Regle anti-derive

Si un visuel pourrait appartenir a une autre agence generique, il n est pas assez Localia.
