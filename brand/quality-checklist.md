# BE-001 - Checklist qualite Localia

Un contenu ne doit passer en `ready` que si tous les points essentiels sont valides.

## Texte

- Le sujet correspond a Localia.
- Le benefice client est clair.
- Le texte Facebook est naturel et professionnel.
- La version Instagram est plus concise.
- Les hashtags Instagram sont pertinents.
- Le CTA est present.

## Visuel

- Le layout respecte un modele Localia.
- Les couleurs restent dans la palette.
- Le titre est lisible sur mobile.
- Les mots importants sont en orange.
- Le visuel principal est premium.
- Les blocs sont arrondis et lisibles.
- Le CTA ou la signature est visible.

## Strategie

- Le contenu n est pas un doublon recent.
- Le sujet apporte de la valeur a une entreprise locale.
- La publication peut mener a une prise de contact.

## Validation finale

Si une seule regle critique echoue, le contenu reste en `draft`.
