# BE-001 - Types de publications Localia

## P01 - Probleme / solution

Expose un probleme courant d une entreprise locale puis presente Localia comme solution.

Exemples :
- invisible sur Google ;
- page Facebook inactive ;
- site internet depasse ;
- perte de demandes clients.

## P02 - Conseil utile

Donne un conseil concret et simple a appliquer.

## P03 - Offre

Presente une offre claire avec benefices et CTA.

## P04 - Avant / apres

Montre la difference entre une presence en ligne faible et une presence optimisee.

## P05 - Checklist

Liste les points a verifier pour ameliorer sa visibilite.

## P06 - Education IA

Explique comment l IA peut aider une petite entreprise sans jargon.

## P07 - Application metier

Montre comment une application simple peut gagner du temps ou eviter des erreurs.

## P08 - Saison / actualite locale

Adapte le message a une periode : ete, rentree, Noel, soldes, vacances, saison touristique.

## Regle

Chaque publication doit avoir un type explicite dans son JSON.
