# Localia - regles visuelles et editoriales

## Regle principale

Les futurs visuels doivent rester dans la meme famille graphique que les visuels deja publies par Localia sur Facebook.

## Si la galerie Facebook est inaccessible

- ne pas inventer une nouvelle direction artistique ;
- appliquer uniquement les codes Localia deja valides ;
- produire un visuel sobre ou un brief detaille ;
- ne pas changer la charte sans validation.

## Codes Localia connus et valides

- Univers : agence digitale locale, professionnelle, moderne, premium.
- Couleurs dominantes : verts Localia, touches orangees ou ambre.
- Style : clair, lisible, technologique mais accessible.
- Cible : commercants, artisans, PME locales, associations.
- Message : visibilite, SEO local, Google, reseaux sociaux, IA, site internet, automatisation.
- Ton : direct, rassurant, oriente resultat.
- Promesse : aider une entreprise locale a etre trouvee et contactee.

## Interdictions

- Ne pas creer un style totalement different.
- Ne pas utiliser de couleurs hors charte sans raison.
- Ne pas surcharger le visuel.
- Ne pas inventer un logo si le logo officiel n est pas disponible.
- Ne pas promettre de resultats irrealistes.
- Ne pas publier un visuel si son style contredit les publications Facebook existantes.

## Format recommande

- Facebook : carre 1080 x 1080 ou vertical 1080 x 1350.
- Instagram : carre 1080 x 1080 ou vertical 1080 x 1350.
- Preference : visuel avec peu de texte, premium, lisible sur mobile.
