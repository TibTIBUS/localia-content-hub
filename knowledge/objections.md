# BE-002 - Objections clients

## Je n ai pas besoin de site

Reponse : un site n est pas seulement une vitrine. C est un point de confiance, une base pour Google et un endroit clair pour expliquer l activite.

## Je suis deja sur Facebook

Reponse : Facebook est utile, mais il ne remplace pas une presence complete. Un client peut chercher sur Google, consulter un site, puis regarder les reseaux sociaux.

## Je n ai pas le temps

Reponse : c est justement le probleme que Localia peut aider a resoudre avec une presence plus structuree et des contenus prepares.

## C est trop technique

Reponse : Localia rend les choses simples. Le client n a pas besoin de comprendre la technique pour profiter du resultat.

## Je n ai pas le budget

Reponse : l objectif est de proposer des solutions adaptees aux petites entreprises, avec un rapport utilite/prix clair.

## Je fonctionne deja au bouche-a-oreille

Reponse : le bouche-a-oreille reste puissant, mais aujourd hui beaucoup de personnes verifient ensuite sur internet avant de contacter.
