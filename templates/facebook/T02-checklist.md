# BE-003 - Template Facebook T02 Checklist

## Usage

Quand le contenu donne des points de verification simples.

## Structure texte

1. Accroche : question ou probleme.
2. Liste de 3 a 5 points.
3. Conclusion : ce que ces points changent.
4. CTA.

## Structure visuelle

Layout recommande : L04_checklist_blocks.

Zones :
- Titre fort.
- 3 a 5 cards arrondies.
- Icone simple par card.
- CTA orange.

## Exemple d angle

Votre entreprise est-elle facile a trouver en ligne ? Verifiez ces 5 points.
