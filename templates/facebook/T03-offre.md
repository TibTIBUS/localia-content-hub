# BE-003 - Template Facebook T03 Offre

## Usage

Quand le contenu presente une offre Localia.

## Structure texte

1. Accroche orientee resultat.
2. Offre expliquee simplement.
3. Benefices concrets.
4. Prix ou condition si applicable.
5. CTA.

## Structure visuelle

Layout recommande : L02_white_premium_card.

Zones :
- Titre tres visible.
- Prix ou promesse en orange si pertinent.
- Blocs benefices.
- Illustration simple.
- CTA bas.

## Regle

L offre doit rester claire, sans surpromesse.
